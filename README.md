# News-Feed
Имитация ленты новостей 
News Feed iOS — демонстрационное приложение постраничной загрузки новостей и аватаров пользователей с поддержкой офлайн-кэширования.
Посты загружаются из сети по 10 штук, если пролистываются все посты, то идет дозагрузка еще 10 постов. Если, при запуске приложения отсутствует интернет, то данные загружаются из CoreData.
В проекте используется:
Clean Architecture
паттерн MVVM
стороннаяя библиотека Alamofire
реакитивный фреймворк Combine
UI фреймворк UIKit
хранение данных CoreData
использование протоколов и инъекция зависимостей
Реализованы: загрузка новостей, хранение данных в CoreData и их чтение, анимация загрузки при запуске приложения, обновление новостей через pull-to-refresh,
пагинация с последующей загрузкой новостей.
